spring.datasource.url -- Connects to PostgreSQL database.
spring.datasource.username - 	PostgreSQL username.
spring.datasource.password -	PostgreSQL password.
spring.datasource.driver-class-name	 - Specifies PostgreSQL JDBC driver.
spring.jpa.database-platform - 	Tells Hibernate to use PostgreSQL syntax.
spring.jpa.hibernate.ddl-auto - 	Controls schema creation (update is safe).
spring.jpa.show-sql - 	Prints SQL  queries in the console.
spring.jpa.properties.hibernate.format_sql - 	Formats SQL for better readability.
